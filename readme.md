В задаче будет автор текст и дата добавления
Задачу надо задавать в json формате
ТЗ: HTTP-сервис "TODO-List" (Golang)1. Функционал
Реализовать сервер, который принимает и отдает задачи через HTTP в формате JSON.
2. Модель данных (JSON)Каждая задача — это структура:
id (int): уникальный номер.author (string): 
кто добавил.text (string): 
суть задачи.created_at (string): 
дата в формате RFC3339.
3. API ЭндпоинтыМетодПутьОписаниеPOST/tasksСоздать задачу. Принимает author и text. Сервер сам ставит id и время.GET/tasksВернуть список всех созданных задач в виде массива.GET/tasks/{id}Вернуть одну задачу по её номеру.4. Критерии приемкиJSON-only: Все ответы должны идти с заголовком Content-Type: application/json.Валидация: Если в POST-запросе пустое поле text — отвечать ошибкой 400 Bad Request.Хранение: На первый раз храни всё в обычном slice (слайсе) в памяти сервера.Потокобезопасность: Обязательно используй sync.Mutex, чтобы сервер не упал, когда два запроса придут одновременно.
